Bot General

Leveling System

  • /l1-level - Check your or another user's level
  • /l2-leaderboard - View top 10 users
  • /l3-setup-level-reward - Set role reward for level
  • /l4-level-reward-show - Show configured rewards
  • /l5-notify-level-msg - Set level-up channel
  • /l6-xp-settings - Configure XP rates
  • /l7-level-config - Configure system behavior

Analytics

  • /a1-dashboard - View analytics dashboard
  • /a2-history - View past weekly reports
  • /a3-generate-snapshot - Generate manual snapshot

YouTube Notifications

  • /y1-find-youtube-channel-id - Find channel ID
  • /y2-setup-youtube-notifications - Setup tracking
  • /y3-remove-youtube-notifications - Remove tracking
  • /y4-list-youtube-notifications - List all configs

Time & Clocks

  • /t1-setup-clock - Create timezone clock
  • /t2-list-clocks - View all clocks
  • /t3-remove-clock - Remove clock config

Smart Reminders

  • /r1-list - View all reminders
  • /r2-delete - Delete a reminder
