Frequently Asked Questions

Common questions about Supporter Bot's features.

Is Supporter Bot free?

Yes! Core features like moderation, leveling, and basic analytics are completely free to use.

Why use the Web Dashboard?

Slash commands cover basic functionality, but the dashboard offers full control and advanced features like visual analytics, detailed configuration, and easier management.

Are level rewards unlimited?

Absolutely. You can set up as many role rewards for different levels as you need without any restrictions.

Can I track unlimited YouTube channels?

Yes, there is no limit. You can track as many creators as you want and send notifications to specific Discord channels.

How do Smart Reminders work?

You can set recurring reminders (e.g., "every 24 hours") for any channel. The bot will automatically tag roles or send messages at your specified intervals.

How do I set up Tickets?

Use the /tt1-setup command to create a ticket panel. Dashboard access allows you to view and manage transcripts of closed tickets.
